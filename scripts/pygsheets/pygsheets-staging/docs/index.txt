`Google Spreadsheets Python API V4`_.

.. _Google Spreadsheets Python API: https://github.com/nithinmurali/pygsheets

A simple, intuitive library which gets most of your work done.

Features
--------

Features:

* Simple to use
* Google spreadsheet api __v4__ support
* Open, create, delete and share spreadsheets using _title_ or _key_
* Control permissions of spreadsheets.
* Extract range, entire row or column values.
* Do all the updates and push the changes in a batch
